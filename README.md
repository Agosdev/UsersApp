## Set up

* FE
  - npm i
  - npm run dev
* BE
  - npm i
  - npm run start

## Tech Stack

* React js + vite
* Typescript
* Nest js
* MongoDB

## Features

* Select a role profile
  - Teacher:
    - Get a list of users
    - Create users (by changing role type)
    - Update users
    - Delete users
  - Student:
    - Pay course with MercadoPago integration: CheckoutPRO API
